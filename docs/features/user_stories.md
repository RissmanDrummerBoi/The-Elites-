User Stories

As a student, I want to log into the system so that I can report maintenance issues.
As a student, I want to submit a complaint so that my problem can be fixed.
As a student, I want to track the status of my complaint so that I know when it will be resolved.
As maintenance staff, I want to view assigned issues so that I can work on them.
As maintenance staff, I want to update the status of a complaint so that progress is visible.
As a residence manager, I want to monitor all complaints so that I can ensure they are handled properly.
