Features List - Residence Maintenance Reporting System
Must Have Features
Student registration and login
Submit maintenance complaints
Categorization of issues (plumbing, electrical, etc.)
Complaint status tracking (pending, in progress, resolved)
Notification system for maintenance staff
Should Have Features
Dashboard showing submitted complaints
Admin panel for managing and assigning issues
Ability to update and edit complaint status
Could Have Features
Student feedback after issue resolution
Upload images of the problem
History of past complaints
